# Repositorio de Estudio de Python - Universidad Nacional Rosario Castellanos

Este repositorio contiene los materiales de estudio, ejemplos de código y ejercicios prácticos para el aprendizaje del lenguaje de programación Python. Ha sido estructurado y diseñado específicamente como un recurso académico de apoyo para los estudiantes de la **Universidad Nacional Rosario Castellanos**.

A través de las diferentes sesiones, encontrarás el código desarrollado en clase para comprender los conceptos fundamentales de programación, construir una base sólida y practicar la resolución de problemas.

## Contenido del Repositorio

El material está estructurado en diferentes sesiones progresivas:

- **Sesión 1 (S1):** Introducción a Python y conceptos básicos sobre lenguajes de alto nivel.
- **Sesión 2 (S2):** Manejo de tipos de datos primitivos (`str`, `int`, `float`, `bool`, `NoneType`), uso de la función incorporada `type()` para inspeccionar clases, cambio de tipo de valor (Type Casting) y ejercicios prácticos sobre variables.
- **Sesión 3 (S3):** Interacción con el usuario a través de la función `input()`, aplicación de operadores matemáticos, lógicos y de asignación, y ejercicios de resolución de problemas con operadores.
- **Sesión 4 (S4):** Control de flujo empleando sentencias condicionales (`if`, `elif`, `else`), evaluación mediante operadores de comparación y magnitud, uso de condicionales anidadas y operador ternario. Finaliza con ejercicios de sentencias condicionales.

## Instrucciones para descargar (clonar) este repositorio en tu PC

Para poder visualizar, ejecutar y modificar los ejercicios en tu propia computadora, es necesario descargar este repositorio utilizando **Git**. Sigue los pasos a continuación:

### Paso 1: Instalar Git y GitHub

Git es la herramienta de control de versiones que permite descargar ("clonar") el código desde GitHub a tu equipo.

- **En Windows:**
  1. Ve a la página oficial de descargas: [git-scm.com/download/win](https://git-scm.com/download/win)
  2. Descarga el instalador correspondiente a tu sistema (normalmente "64-bit Git for Windows Setup").
  3. Ejecuta el archivo descargado y sigue las instrucciones del instalador. Puedes dejar todas las opciones predeterminadas y hacer clic en "Next" hasta finalizar.
- **En macOS:**
  1. Abre la aplicación **Terminal**.
  2. Escribe el comando `git --version` y presiona Enter. Si Git no está instalado, el sistema te mostrará una ventana pidiendo que instales las "Command Line Tools" de Apple. Acepta la instalación.
- **En Linux (Ubuntu/Debian):**
  1. Abre tu terminal y ejecuta el comando: `sudo apt-get install git`.

### Paso 2: Abrir la terminal de comandos

- **Windows:** Presiona la tecla `Windows`, busca **Git Bash** (la herramienta que acabas de instalar) y ábrela. También puedes usar `cmd` o `PowerShell`.
- **macOS / Linux:** Abre la aplicación llamada **Terminal**.

### Paso 3: Clonar el repositorio

En tu terminal, debes moverte a la carpeta donde deseas guardar los archivos de estudio. Por ejemplo, si quieres guardarlo en la carpeta de Documentos, escribe el siguiente comando y presiona Enter:

```bash
cd Documentos
```

A continuación, ejecuta el comando de clonación. Asegúrate de reemplazar _URL_DEL_REPOSITORIO_ por la liga real (URL) de GitHub de este proyecto:

```bash
git clone https://github.com/ikaruslab1/Clases-python
```

### Paso 4: Acceder a los archivos locales

Una vez que el proceso de descarga termine, se habrá creado una nueva carpeta en tus Documentos con todo el código. Ingresa a la carpeta ejecutando:

```bash
cd nombre-del-repositorio
```

_(Nota: Reemplaza "nombre-del-repositorio" por el nombre exacto de la carpeta que se acaba de descargar)._

¡Listo! Ya tienes todos los archivos .py de las sesiones almacenados de forma local en tu computadora. Ahora puedes abrirlos con tu editor de código preferido (como Visual Studio Code) para revisar los apuntes, completar los ejercicios y ejecutar el código.
